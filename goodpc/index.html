You DON'T NEED a good PC

here’s a simple way to have more money. for free!

What do you need?

First, we need to specify how we use computers. Here are some of the more performance-heavy things that people use PCs for:

  • Audio / Video / Image Editing
  • Gaming
  • Web Browsing
  • Text Editing / Programming

You probably could do all of these things on a 400$ box. But how?

Lower you expectations.

Yes, really. The most important thing is to lower your expectations and stop fearing change. You need to basically realize that you don’t need to play - i don’t know, Soyberpunk 2077 with raytracing.

But don’t fear, it is almost always the programmers fault!

For example, let’s take Minecraft. The OG edition of the game is written in java. Because it was written in java in 2009, the game code runs mostly single-threaded and isn’t optimized. When microsoft decided to make an UWP version of minecraft for windows 10, the xbox and mobile devices - they decided to rewrite the engine in C++. The results are that in the java version of the game, the max. world render distance is 64 (with optimization mods!). In the C++ version it’s the default.

Still, it’s a small issue compared to…

mUh HeCkIn WeB bRoWsErS1!1!!

I think it is safe to say it: the web is beyond repair. The sites are bloated, the browsers are bloated, the languages are bloated. EVERYTHING. IS. BLOATED.

I genuinely think that without a decent PC you can’t browse the internet comfortably. For example, Firefox starts up on my PC (it’s pretty beefy, even if the OS is installed on an HDD) in around 5 - 10 seconds. In comparison - starting up Firefox on my Thinkpad x230 takes 30 seconds. The worst thing is that there aren’t any browsers that are really faster and have good website support, and chromium has the same issues.

(yes, i know that surf exists, dont @ me)

Even if web browsers were more optimized, they would still have high system requirements due to the web technologies themselves. Websites really shouldn’t compile 25 megabytes of javascript. If your site requires javascript to function - it probably shouldn’t be a website. Web developers should focus more on stripping down the javascript, not on making the site unusable.

So how to actually have a better PC for free

  1. Install linux

Yes, it is required, and you won’t escape from it. You can play almost every game on linux already (from my experience), so there aren’t any reasons to not jump ship. Windows always was the most bloated OS on the market, and it’s mere existence is constantly pushing hardware manufacturers to release overpriced, power-hungry CPU’s and GPU’s. As for my recommendation - if you want a huge performance boost, just install linux. I would suggest Linux Mint MATE edition.

  1. (OPTIONAL) Install BSD

if you got the balls to do it. do it.

  1. Optimize your OS.

on windows you really can’t truly optimize your system, so i’ll just explain what to do on UNIX-based systems.

  • Install a more lightweight graphical environment. For example if you use KDE, install XFCE. If you use XFCE, install a tiling window manager etc. Configuring a tiling WM can be hard, but it will always be way faster than GNOME / KDE.

if you don’t mind wayland you could even use (for gaming) the gamescope compositor made by valve for the steam deck.

  • Remove unnecessary services running in the background Get rid of plymouth, sshd and other stuff you don’t need, you could theoretically not use a display manager and instead use cli login with .bashrc xorg autostart. It will basically save you ram, and maybe even CPU usage.
  1. Stop running other aplications while gaming.

It will greatly increase availible RAM. Needed for 4GB (or less) ram systems.

  1. Lower your resolution in games (if you can)

AMD’s FSR works on linux without any problems (you can enable it via lutris or in gamescope). if you have a 1920x1080 screen you could just upscale from 1280x720 or 1600x900. It will increase performance with little loss of visual quality.

  1. Don’t enable the FPS counter

Here’s a little psychological trick. Just stop caring about the FPS and find the settings that you could play on, without setting the barrier to 60 fps or 30. Most people could probably play comfortably in 48 fps, and in some games even 24 fps without even knowing it.

  1. (OPTIONAL) Sell your PC and buy “a better one”

It sounds stupid, but with experience you could buy some way better parts for the same price. You could basically in a few weeks get a 20 - 50% performance boost just by picking your PC parts better. Also you can just add a couple of bucks to your budget and buy a objectively better computer with a very small price increase (let’s say 20 - 50$).

Start using software that actually cares about performance.

There is a ton of FOSS software that runs a lot better than all of it’s proprietary counterparts. You’d be suprised that you could edit videos, do VFX, produce music and code comfortably on a 250$ box.

Conclusion

Use your 10 dollar PCs, install gentoo and have fun getting max performance out from you computers!
