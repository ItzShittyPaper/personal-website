kektech: The 'no-compromises' 2D multimedia engine

kektech is a simple game engine that’s focused on performance. Built on top of the reliable SDL library. kektech provides an easy to use API that makes programming in C fun and efficient, so you can focus on your project while the engine handles everything else.

while the project is a work-in-progress for now, most things are already done, like:

  • textures and a primitive rendering system suitable for basic geometry
  • a simple input system that detects key presses and releases
  • TTF font support
  • tilemapping using a custom map format, datadesk
  • music and SFX playback (WAV, OGG, tracker modules like XM or IT) and
  • a simple GUI library, kkui

The engine is optimised for UNIX systems, but should work on Windows platforms just fine, since it doesn’t use any libs from POSIX.
