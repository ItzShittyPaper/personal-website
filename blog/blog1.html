Your Website Sucks! (probably)

Almost everybody knows that the web is beyond repair. The internet is filled with ads, javascript and bloat. I usually try to make my websites minimal, without overdesigning it and instead focusing on the content, because that's why people went there. Here are some rules that i follow to avoid bloat:

No Javascript

Javascript is a gateway drug for web developers. It is a full scripting engine in your browser. Almost everyone that learns it spends a couple of days just having fun with it's possibilities. Then, when making a serious project, most of them think "I can do this way easier with javascript". They use it even more, almost exclusively. Because of it sites made by them are usually slow and sluggish. For example Google Docs, a popular web-based office suite downloads around 25 megabytes of javascript code that needs to be there so the web app could work. That means it takes around 5 seconds to load it in a mobile 4G internet, which is still heavily used. I usually don't use javascript whatsoever, it should be used only in certain situations, like for example a donation system.

Everything needs to be in plain sight

What does that mean? Basically removing all the sidebars, navbars, context menus and other bullshit. Pages should be sorted by the topic that they represent. Everything on the topic of a page is there, completely visible to the viewer.

An RSS feed

For me, every webpage on the internet should have it's own rss feed. It is a decentralized, uncensored on-line new protocol. You can read html-based news from a RSS reader. If you do some blogging, it is the best option in my opinion.

The Conclusion

I use these three simple rules to make my website clean and with as little bloat as possible. Some people might think that this is a bit extreme, but looking at the most popular sites of the internet - they are actually standard. For example the old google. The first version of google.com was just a simple box that you type things to, next you click the button below and the website shows you a list of links based on what you typed into the box. Wikipedia was just a simple libary of markdown-style files. And the first version of youtube was just a list of html video tags.

I hope this pile of text was useful to somebody.
